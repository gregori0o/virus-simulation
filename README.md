# Virus Simulation

## Requirements

Python 3.10  
[requirements.txt](requirements.txt)

## Developing

```shell
git clone https://github.com/gregori0o/virus-simulation.git virus-simulation
cd virus-simulation
virtualenv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
pre-commit install
```

## Running app

```shell
python src/app.py
```
